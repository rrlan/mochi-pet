# 杨百乐 — 示例形象包 🐈

这是 [@rrlan](https://github.com/rrlan) 的猫 **杨百乐** 的一套 Mochi 形象包，开箱即用，
欢迎直接拿去用。

| 文件 | 状态 |
| --- | --- |

| `slack.png` | 摸鱼 |

## 直接用

这套已**随 App 内置**，无需手动拷贝：打开 🍡 → 形象 → **选择形象…**，在缩略图面板里点一下「杨百乐」就换上了；点「默认 Mochi」随时换回纯代码的薄荷史莱姆。

想拿这套图改成自己的形象包，照着上面的文件名结构放图，再用菜单 **形象 → 导入形象包（文件夹）…** 选你的文件夹即可。
