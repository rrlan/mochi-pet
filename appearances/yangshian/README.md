# 杨十安 — 示例形象包 🐈

[@rrlan](https://github.com/rrlan) 的另一只猫 **杨十安**（棕色虎斑），和 [`../yangbaile/`](../yangbaile/) 一样**随 App 内置**、开箱即用。

| 文件 | 状态 |
| --- | --- |
| `companion.png` | 陪伴 · 待机（主形象） |
| `work.png` | 工作中（agent 在跑） |
| `rest.png` | 休息 · 睡觉 |
| `slack.png` | 摸鱼 |
| `drag.png` | 被拖动 |
| `walk/frame_00–01.png` | 走路逐帧动画 |

## 直接用

🍡 → **形象…** 打开选角面板：点缩略图把它设为**主猫**，或点爪印让它作为**氛围猫**出来逛。

> 这些图片随本仓库一同以 MIT 授权，随意使用。想做自己的形象包，照着这个文件名结构放图即可。
